a minimalistic Excel-like data grid editor for HTML, JavaScript & jQuery

Editor templates

This example how to use template and editor_template to use your own HTML as cell renderer/editor in a simple way.

  • Year is a column with input type="number" editor
  • Chassis color is a column with select renderer and editor
  • 4WD is a column with input type="checkbox" renderer and editor
